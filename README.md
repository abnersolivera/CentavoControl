# CentavoControl

CentavoControl é uma aplicação para controle de finanças pessoais, focada em facilitar a organização das suas contas a pagar e o gerenciamento de cartões de crédito.

## Funcionalidades

- Cadastro e acompanhamento de contas a pagar
- Controle de cartões de crédito
- Visualização de despesas e receitas
- Relatórios financeiros para melhor tomada de decisão

## Como usar

1. Clone este repositório
2. Instale as dependências necessárias
3. Execute o projeto conforme as instruções da documentação técnica

## Objetivo

Auxiliar usuários no controle financeiro pessoal, evitando esquecimentos de pagamentos e promovendo uma melhor saúde financeira.
